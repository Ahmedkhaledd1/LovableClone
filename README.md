# 🚀 LovableClone

LovableClone is an AI-powered project builder that transforms a simple natural language prompt into a functional web application.

It leverages a multi-agent workflow (planning → architecture → implementation) to automatically generate frontend projects, including HTML, CSS, and JavaScript files.

---

## ✨ Features

- 🧠 Convert plain English prompts into working web apps  
- 📋 Intelligent task planning using structured agents  
- 🏗️ Automated architecture and file organization  
- 💻 Code generation for frontend (HTML, CSS, JavaScript)  
- 🔄 Iterative file creation and updates  
- 📁 Outputs a ready-to-run project inside `generated_project/`  

---

## 🛠️ Tech Stack

- **Python** – Core backend logic  
- **LangChain** – LLM orchestration  
- **LangGraph** – Multi-agent workflow management  
- **Groq** – Fast LLM inference  
- **Pydantic** – Data validation and structured schemas
